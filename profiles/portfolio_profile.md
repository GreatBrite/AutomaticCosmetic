# Cosmetology Automation Profile

Use this profile when judging work for the Automatic Cosmetic project.

The project goal is an operations system for a cosmetologist.

Useful fit signals:

- Avito client dialogue automation.
- Telegram admin workflows for appointments and client notes.
- YCLIENTS service, slot, client, appointment, and note operations.
- Editable knowledge base for services, prices, contraindications, preparation, and aftercare.
- Care and upsell planning after visits.
- Human handoff for photos, complaints, medical uncertainty, and unusual requests.

Avoid enabling live replies or live YCLIENTS mutations without a controlled real-scenario test.
